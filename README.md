# Videocall App
A video chat/meeting webapp using WebRTC and WebSockets. video call+ white board.
# Features
1. Video Calls
2. Chatbox
3. Whiteboard
4. Screensharing
# Getting Started
to test the app on your local server:
Run `npm i` to install all the dependencies. 
Run `nodemon` to start the server.
# Tech Stack
1. React
2. NodeJS
3. Express
4. WebSockets
